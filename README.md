### Dependencies

- Python 3.8
- Poetry


### pre-commit

Install

``` bash
pre-commit install

pre-commit install -t pre-push
```

Run locally
``` bash
pre-commit run --all-files
```

### Test
Run tests
``` bash
pytest tests/
```
